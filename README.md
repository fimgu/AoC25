## Structure

Each day's solution is contained within its own folder (e.g., `day1/`, `day2/`).
Inside each folder, you'll typically find:
- `dayX.cpp`: The C++ source code for the solution.
- `input.txt`: The puzzle input.
- `test.txt`: Example test cases.

To compile and run a specific day's solution (assuming `g++` is installed):

```bash

./day1
```
